# Supply Chain Analytics Dashboard using Power BI

## Project Overview

This project presents an interactive Supply Chain Analytics Dashboard built using Power BI. The dashboard provides insights into revenue performance, inventory management, supplier efficiency, manufacturing costs, logistics operations, and product quality.

The objective of this project is to transform raw supply chain data into actionable business insights that support strategic decision-making.

## Key Features

### Executive Overview
- Total Revenue Analysis
- Units Sold Tracking
- Product Performance Analysis
- Location-wise Revenue Distribution
- Transportation Mode Analysis
- Defect Rate Monitoring

### Inventory & Supplier Performance Analysis
- Inventory Level Monitoring
- Supplier Revenue Comparison
- Manufacturing Cost Analysis
- Stock Status Distribution
- Lead Time Evaluation

## Business Insights

- Skincare products generated the highest revenue and sales volume.
- Mumbai contributed the highest revenue among all locations.
- Rail transportation accounted for the largest revenue share.
- Supplier 1 generated the highest revenue and manufacturing volume.
- Inventory levels remained balanced across product categories.
- Product defect rates remained low, indicating effective quality control.
